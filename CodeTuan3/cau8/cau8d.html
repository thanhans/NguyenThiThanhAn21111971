Border Radius

Sytax:

border-radius:

(top-left) (top-right) (bottom-right) (bottom-left);

Example:

border-radius:

0px 30px 70px 20px;

For displaying round corners Border Radius property is used.
IE9 and Opera use the same code. For Firefox a prefix

"-moz-"

need to be added the code. And for Webkit Browesrs

"-webkit-"

needs to be added before the code.
